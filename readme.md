## Getting Started
### Prerequisites
The following is required to build and run GameBruh
```
SDL2
```

### Building and Running (Mac)
you can build and run by downloading the source code and running the following in the terminal:
```
cd <path of GameBruh>
Make GameBruh
./GameBruh
```
